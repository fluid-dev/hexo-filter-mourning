# hexo-memorial-day

Grey your hexo site on the memorial day

哀悼日专用 Hexo 插件，开启后全站灰色

## Installation

```shell
$ npm i hexo-memorial-day --save
```

## Config

In your site's `_config.yml`:

```yaml
memorial_day:
  # Better to close it when it is not used
  enable: true
  # Specified date will be greyed
  day: 2020-04-04
```
